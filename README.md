| :-: | :-: | :-- |
| PORT | 8888 | Webhook の port |
| LOG_LEVEL | all | |
| TOKEN | | Line の channelAccessToken |
| SECRET | | Line の channelSecret |
| ROOT | | プログラムのルートディレクトリ (/app 等) |
| BACKEND_SERVER_URI_BASE | | バックエンドサーバの URI (https://tut-php-api.herokuapp.com/api/v1 等)
